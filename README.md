# README

## Name

grn2drn

## Description

Grn2drn is a command to convert a *.grn dump file to a JSONs file for Droonga.

## Install

```
% gem install grn2drn
```

## Basic usage

```
% grn2drn --dataset Droonga /path/to/grn/dump/file
```
